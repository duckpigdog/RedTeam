### /bin/bash 指令提权

```bash
sudo /bin/bash -p
```

`sudo`

- 用 root 权限执行命令
- 这里非常重要，因为你要打开的是一个特权 shell

`/bin/bash`

- 指定系统自带的 bash 可执行文件，而不是当前目录的 `./bash`

`-p` 参数

- 保留 **有效 UID/GID**（privileged）：
  - 默认情况下，`sudo bash` 会重置一些环境变量和权限
  - `-p` 保留原始特权，用于 **提权 shell** 或 **root 克隆账户**
