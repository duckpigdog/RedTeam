### MSF auxiliary 模块 Telnet 爆破

又想到了 23 端口号 telnet 服务，试着打开 telnet 服务

```
set command sc config tlntsvr start= auto
set command net start telnet
```

```
use auxiliary/scanner/telnet/telnet_login
```
