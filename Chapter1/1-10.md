### Linux 服务信息收集

**判断是否是在 docker 容器中**

```sh
ls -la /.dockerenv
```

![](https://pic1.imgdb.cn/item/68cbb8eec5157e1a8816b4f3.png)

**查看进程**

```sh
ps -aux
```

![](https://pic1.imgdb.cn/item/68cbb931c5157e1a8816b7be.png)

```sh
crontab -l
```

![](https://pic1.imgdb.cn/item/68cbb954c5157e1a8816b871.png)

```
uname -a
```
